# ReportFlow v0.1.0

Initial public release of ReportFlow.

## Highlights

- layout engine with deterministic pagination for enterprise reports
- long tables with repeated headers and row-safe pagination
- charts, KPI cards and watermark support
- theming via design tokens for multi-brand report templates
- CLI for rendering reports from declarative templates and JSON data

## Packages

- `@reportflow/core@0.1.0`
- `@reportflow/cli@0.1.0`
